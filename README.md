## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YashAi-Tech/portfolio.git
```

Go into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
